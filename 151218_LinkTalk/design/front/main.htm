링크톡은?

링크톡은 기업 활동에서의 가장 핵심적 요소인 ‘협업'과 ‘커뮤니케이션’을 장소나 기기에 제한 없이 쉽고 빠르게 수행하여 조직 간의 정보 공유를 이끌어 내는 기업 전문 솔루션 입니다.

LinkTalk 도입효과

Quick Communication
  • • 모바일 Push 알림, 다자간 실시간 대화의 기능으로 정보를 놓치지 않음
  • • 채널 기능을 활용 사내 정보를 빠르게 처리 가능
  • • 조직도 기반 검색으로 동료를 직관적 검색 가능
Smart Work
  • • 통합검색 기능으로 원하는 정보를 한번에 검색
  • • Anytime, Anywhere, Any Device 솔루션으로 언제 어디서나 실시간 업무처리
  • • 동료그룹 기능을 이용하여 반복적인 작업을 줄임
Efficient Collaboration
  • • 채널 기능을 활용하여 기존 사내 정보를 링크톡으로 공유가 가능
  • • 편리한 파일 UX 제공하여 자료 공유 편의 제공

LinkTalk의 특장점

  • Communication - 실시간 대화와 알림

    업무 진행 시 즉각적인 알림과 실시간 모바일 메신저는 업무를 빠르고 효율적으로 처리할 수 있도록 도와줍니다.

  • Collaboration - 효율적인 협업 환경 제공

    링크톡 채널 기능을 활용하여 현재 사용중인 사내의 시스템 정보를 링크톡 모바일안에서 사용할 수 있습니다. 더불어 자료 등록 또는 회신을 받을 경우에는 모바일 알람 기능도 연동 됩니다.

  • Sharing - 편리한 자료 공유

    대화방에서 자료의 송/수신을 직관적인 UX를 도입하여 보다 쉽고 편리하게 공유할 수 있으며 통합검색 기능을 제공하여 대화나 첨부파일을 한번에 검색할 수 있습니다.

  • Flexible System - 쉽고 빠른 시스템 적용과 확장

    링크톡는 기획 초기부터 Legacy system 과의 연동 적용을 고려하여 개발하였기 때문에 사내 정보를 링크톡에 쉽게 적용할 수 있습니다. 또한, 사용자 증가에 따른 서버를 병렬로 안전하게 증설할 수 있으므로 대량 사용자의 동시 접속을 원활하게 지원 합니다.

  • Multi-Device - 언제 어디서든지 업무처리

    링크톡의 온라인 PC와 모바일 Smartphone, Tablet PC 등 모든 단말기에서의 실시간 연결 기능은 커뮤니케이션과 업무처리 효과를 극대화합니다

링크톡의 주요 기능

링크톡는 실시간 커뮤니케이션을 기업의 업무처리 프로세스에 적용하여 편하고 빠르고 편리한 업무처리를 제공합니다.

  • 편리한 커뮤니케이션 동료와 대화 및 정보 공유를 편리하게 할 수 있는 UX 및 기능을 제공하여 편리하고 빠른 업무처리를 지원
  • 사내 정보와의 유기적 결합 사내 조직도와의 동기화를 지원하며 사용자에게 직관적인 조직도 화면을 제공
    채널 기능을 제공하여 사내의 정보를 링크톡에서 공유 가능 함
  • 편리한 통합 검색 기능 여러 대화방 내의 대화 내용 또는 첨부파일에 대해서 문장을 한번에 통합으로 검색
  • 통신 및 내부 정보 보호 암호화 통신(ARIA, WSS, HTTPS)을 사용하여 네트워크상에서 정보를 보호하며
    모바일 기기의 로그인 인증과 더불어 기기 인증을 통해서 제3자의 접근을 금지
  • Multi-Browser & Multi-Device 지원 IE, Chrome, Safari, Firefox, Opera 등의 멀티 브라우저 지원과 PC 및 모바일(iOS, Android) 앱 지원
